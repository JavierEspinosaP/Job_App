## Installation
***
A little intro about the installation. 
```
$ git clone https://github.com/Nachoooooooo/Job_App.git
install this modules
├──bcryptjs@2.4.3
├── cors@2.8.5
├── cowsay2@2.0.4
├── dotenv@16.0.1
├── express@4.18.1
├── helmet@6.0.0
├── heroku@7.63.0
├── jsonwebtoken@8.5.1
├── mongodb@4.9.0
├── mongoose@6.5.3
├── morgan@1.10.0
├── node-fetch@2.6.7
├── nodemailer@6.7.8
├── nodemon@2.0.19
├── pg@8.8.0
├── postgre@0.1.8
├── pug@3.0.2
├── puppeteer@17.1.2
├── stack@0.1.0
├── swagger-jsdoc@6.2.5
└── swagger-ui-express@4.5.0
$ cd ../path/to/the/file
$ npm install
$ npm start
```
Side information: To use the application in a special environment use ```npm start``` to start
